My Journey in Learning Programming

Programming has been a passion of mine for the past three years. I started my journey by learning languages like C, C++, and Java, which formed the foundation of my skills. As a third-year Computer Engineering student at Al-Quds University in Palestine, I’ve been deeply immersed in coding. These languages have taught me not only the technical aspects of programming but also the importance of logical thinking and problem-solving.

My Current Project: Online Exam System

Currently, I am working on a significant project: an online exam system. This project is part of my effort to improve my Java skills, a language I have grown to appreciate for its versatility and widespread use. The goal of this system is to create a platform where exams can be administered and graded online, providing both students and teachers with a smooth experience. I’ve faced several challenges along the way, particularly in handling user input validation and security, but each challenge has been an opportunity to learn and grow as a developer.

My Passion for Integrating AI in Web Development

One area I am incredibly passionate about is the integration of artificial intelligence (AI) into web development. The possibilities AI offers for making websites smarter and more interactive are endless. That’s why I’m currently taking a Machine Learning with Python course at Al-Raja International. Through this course, I’ve been learning how to use machine learning algorithms to create more personalized and intelligent web experiences. My goal is to build websites that can adapt to user behavior and offer more tailored solutions using AI.

Continuous Learning: My Current Courses

Learning is a never-ending journey, especially in the field of technology. To enhance my skills, I am enrolled in two key courses:

  • Udacity Front-End Web Development: This course is helping me build a strong foundation in HTML, CSS, and JavaScript to design visually appealing and functional websites. It runs for 4 months, ending in February 2025.
  • Machine Learning with Python at The Hope International: This course is giving me hands-on experience in implementing AI solutions using Python, and it will finish by the end of November 2024.

Both courses are sharpening my skills, preparing me to bridge the gap between web development and AI integration.

My Future in Web Development
